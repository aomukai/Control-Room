# World Building: Neo-Seattle

## Overview
Set in 2087, Neo-Seattle is a sprawling megacity built on the ruins of old Seattle.

## Technology
- Neural interfaces common but regulated
- AI assistants ubiquitous
- Holographic displays replace most screens
- Flying vehicles restricted to commercial use

## Social Structure
- Corporate zones (wealthy, controlled)
- Citizen districts (middle class, monitored)
- The Undergrowth (poor, lawless, vibrant)

## Weather
Still rains constantly. Some things never change.
